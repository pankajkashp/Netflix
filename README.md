## ✨ About The Project
Ever wondered how the magical Netflix interface ticks? This project is a lovingly crafted frontend clone aiming to recreate the authentic **Netflix** browsing experience. From the dark, sleek typography to the buttery-smooth hover effects on movie thumbnails, it's all here.

### 🌟 Key Features
- 🎬 **Hero Banner**: Eye-catching main feature display with interactive 'Play' and 'More Info' buttons.
- 📱 **Fully Responsive**: Adapts seamlessly from massive desktop monitors down to mobile screens.
- 🎨 **Dynamic Hover States**: Layout elements and movie cards scale and transform smoothly.
- 🔒 **Login Interface**: Functional design implementations mirroring the real Netflix authentication flow.

## 🚀 Getting Started

No complex build steps. No `npm install`. Just pure, unadulterated web technologies.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/pankajkashp/Netflix.git
   ```
   
2. **Open the magic:**
   Navigate into the downloaded folder and double-click `netflix.html` to open it strictly in your favorite browser.

## 🧠 What I Learned
Building this clone was a fantastic deep dive into:
- Advanced CSS Flexbox & Grid layouts.
- Absolute positioning and z-index mastery for overlapping UI elements (like gradient screen overlays).
- Bringing static pages to life with JavaScript event listeners and DOM manipulation.
