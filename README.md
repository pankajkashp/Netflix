# Netflix Clone

A responsive Netflix frontend clone project built using HTML and CSS.

## Features
- Netflix Homepage UI Layout
- Responsive design (if applicable)
- Interactive buttons and hover states
- Custom styling for navigation and features 

## Tech Stack
- HTML5
- CSS3

## Getting Started
To view this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/pankajkashp/Netflix.git
   ```
2. Open `netflix.html` in your preferred web browser.
